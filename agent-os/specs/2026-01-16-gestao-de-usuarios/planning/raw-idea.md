# Raw Idea: Gestão de Usuários (User Management)

**Feature Name:** Gestão de Usuários (User Management)

**Description:**
CRUD de usuários com atribuição de roles, hospitais vinculados e preferências de notificação.

## Context
This is part of SIDOT - a system for organ donation notification. This feature will allow:
- Create, read, update, delete users
- Assign roles (operador, gestor, admin)
- Link users to specific hospitals
- Configure notification preferences per user
