# Raw Idea: Notificação SMS

**Feature Name:** Notificação SMS

**Description:**
Integrar com gateway SMS (ex: Twilio, Zenvia) para envio de alertas para celulares da equipe de plantão.

## Context

This is part of SIDOT - a system for organ donation notification. This feature will:
- Integrate with SMS gateway provider (Twilio or Zenvia)
- Send SMS alerts to on-call team members
- Work alongside existing email and dashboard notifications
- Be triggered by the same notification system that sends emails
