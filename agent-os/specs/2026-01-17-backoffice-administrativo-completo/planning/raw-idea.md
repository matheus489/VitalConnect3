# Raw Idea: Backoffice Administrativo Completo

## Feature Description

Uma interface de administração completa para a plataforma SIDOT onde administradores podem: gerenciar tenants (criar, editar, configurar), gerenciar usuários de todos os tenants, configurar elementos visuais e branding por tenant, gerenciar hospitais, regras de triagem, configurações do sistema, visualizar logs e métricas globais, e ter controle total sobre a plataforma.
