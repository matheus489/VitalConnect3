# Raw Idea: Gestão de Plantões (Shift Management)

**Feature Name:** Gestão de Plantões (Shift Management)

**Description:**
Cadastro de escalas de plantão com horários e responsáveis, direcionando notificações para a equipe correta do turno.

## Context
This is part of SIDOT - a system for organ donation notification.

## Key Capabilities
This feature will allow:
- Create and manage shift schedules
- Define time slots and responsible staff for each shift
- Route notifications to the correct on-call team based on current shift
- Support multiple hospitals with different shift configurations
