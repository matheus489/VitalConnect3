# VitalConnect MVP - Ideia Inicial

## Descrição da Feature

MVP completo do VitalConnect incluindo:

1. Detecção automática de óbitos via integração com sistemas hospitalares
2. Triagem inteligente para elegibilidade de doação de córneas
3. Sistema de notificações em tempo real para equipes de captação

## Contexto do Produto

O produto é uma solução GovTech de middleware para Centrais de Transplantes e Bancos de Olhos, focando na janela crítica de 6 horas para captação de córneas.
