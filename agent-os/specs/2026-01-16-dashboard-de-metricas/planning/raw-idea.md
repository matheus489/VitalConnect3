# Dashboard de Métricas - Raw Idea

**Feature Name:** Dashboard de Métricas

**Description:**
Tela com gráficos de notificações por período, tempo médio de resposta, taxa de conversão (notificação → captação) e ranking por hospital.

## Context
This is part of SIDOT - a system for organ donation notification. This feature will:
- Display charts showing notifications over time
- Show average response time metrics
- Track conversion rate from notification to successful capture
- Rank hospitals by performance
- Provide visual insights for managers
