# Raw Idea: Integração Real com PEP

## Feature Name
Integração Real com PEP

## Description
Adaptar o listener para conectar a um banco de dados hospitalar real (PostgreSQL/MySQL/Oracle), com mapeamento configurável de campos.

## Context
This is part of SIDOT - a system for organ donation notification. This feature will:
- Connect to real hospital Electronic Health Record (PEP) databases
- Support multiple database types (PostgreSQL, MySQL, Oracle)
- Allow configurable field mapping (different hospitals have different schemas)
- Replace the simulated death listener with real hospital integration
