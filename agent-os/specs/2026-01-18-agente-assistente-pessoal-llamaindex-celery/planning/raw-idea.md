# Raw Feature Idea

**Feature:** Agente assistente pessoal para plataforma VitalConnect usando LlamaIndex e Celery - tirar dúvidas, fazer ações, integrado com a plataforma

## Description

This feature involves creating a personal assistant agent for the VitalConnect platform that:
- Uses LlamaIndex for intelligent question answering
- Uses Celery for asynchronous task processing
- Can answer user questions (tirar dúvidas)
- Can perform actions (fazer ações)
- Is integrated with the VitalConnect platform
